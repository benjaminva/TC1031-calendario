Proyecto Avance 1 Algoritmos de Ordenamiento

En este proyecto vas a almacenar los datos, a ordenar los datos y posteriormente a acceder a ellos de manera eficiiente.

list Instrucciones

Selecciona un reto o problema que involucre el tener que acceder una serie de datos. Crea tu programa haciendo uso de las buenas prácticas de programación y modularidad que has aprendido en materias anteriores.

En este avance el objetivo es que dentro de un programa funcional se puedan organizar los datos almacenados.

Puedes implementar mecanismos para cargar la información a tu programa y filtros para ordenar y deplegar la información de diferentes formas o usando diferentes atributos de los objetos.

Crea un repositorio en github, y sube en el tu proyecto este primer avance.

  • SICT0301: Evalúa los componentes

    Hace un análisis de complejidad correcto y completo para los algoritmos de ordenamiento usados en el programa.

    SICT0302: Toma decisiones

    Selecciona un algoritmo de ordenamiento adecuado al problema y lo usa correctamente.
