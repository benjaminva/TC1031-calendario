Integral A Avance 3 Árboles

En este proyecto vas a almacenar los datos, a ordenar los datos y posteriormente a acceder a ellos de manera eficiiente.

list Instrucciones

En este avance el objetivo es almacenar los datos en un árbol para su rápido acceso.

Haz los cambios pertinentes a tu proyecto para hacer uso de forma util de un árbol.

  • SICT0302B: Toma decisiones

    Usa un árbol adecuado para resolver un problema, puede ser cualquiera de los árboles vistos en el curso, lo lo importante es que se demuestre que se usa forma útil el árbol para extraer y guardar información.

  • SICT0301B: Evalúa los componentes

    * Presenta Casos de Prueba correctos y completos para todas las funciones y procedimientos del programa., se espera documentación limpia y clara de cada función del programa, las pruebas se pueden hacer en un archivo pruebas.cpp con un main separado de pruebas como el que usan en sus actividades formativas.

    * Hace un análisis de complejidad correcto y completo para todo el programa y sus compenetes., se espera documentación limpia y clara de cada función del programa, asi como de los metodos de acceso a la estrcutura y operaciones como inserción, búsqueda, borrado que se estén usando. Este análisis se debe presentar en el .readme del repositorio. Recuerda que el readme se escribe en markdown

  • SICT0303B: Implementa acciones científicas

    * Implementa mecanismos para consultar información de las estructuras correctos y útiles dentro de un programa., funciones o procedimientos que permitan al usaurio del programa consultar la información guardada en las estrcuturas de datos, es decir, hacer búsquedas u obtener valores específicos (no solo imprimir toda la estructura).

    * Implementa mecanismos de lectura de archivos correctos y útiles dentro de un programa. Usar de manera inteligente archivos de texto para cargar datos iniciales a la estructura o al programa, para no tener toda la información en el código fuente y que se pueda reutilizar el programa con diferentes archivos de texto.

    * Implementa mecanismos de escritura de archivos correctos y útiles dentro de un programa. Usar de manera inteligente archivos de texto para guardar los cambios hechos a la estructura o a las variables del programa, para no perder el trabajo que ha avanzando el usuario.

  • Nota: * Esta evidencias se puede presentar en esta actividad o en la actividad integral B no es necesario presentarla en ambas repositorios.
