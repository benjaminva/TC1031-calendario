Proyecto entrega final

En este proyecto vas a almacenar los datos en un grafo, vas a buscarlos con un algoritmo de búsqueda y a guardar los datos principales en una tabla hash o diccionario para acceder a ellos de manera eficiiente.

list Instrucciones

Selecciona un reto o problema que involucre el tener que represntar los datos en un grafo, ya sea para representaciones simbólicas o espaciales. Posteriormente tendrás que reorrer el grafo usando algoritmos de búsqueda y podrás usar una tabla hash para poder mapear valores del grafo. Crea tu programa haciendo uso de las practicas de programación y modularidad que has aprendido en materias anteriores.

En este avance el objetivo es almacenar los datos en un grafo.

Crea un repositorio en github, y sube en el tu proyecto este primer avance.

  • SICT0302B: Toma decisiones

    Usa grafos para hacer analisis de información, guarda la información de problemas o conceptos que tengan multiples relaciones, como las conecciones de una red social, de una pandemia, de un espacio físico, de los estados de algún objeto, etc...

    Usa un algoritmo de búsqueda en grafos adecuado para resolver un problema encuentra una ruta de conecciones o acciones dentro del grafo.
